# Tic-Tac-Toe (C#)

A console-based Tic-Tac-Toe game built in C# using Visual Studio.

## Features

- Two-player gameplay
- Input validation
- Win detection
- Draw detection
- Play again option
- Clean object-oriented design

## Technologies

- C#
- .NET
- Visual Studio

## Concepts Demonstrated

- Object-Oriented Programming
- Arrays
- Loops
- Methods
- Input Validation
- Modular Design
